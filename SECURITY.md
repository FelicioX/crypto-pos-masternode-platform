# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately before disclosing it publicly.

### Contact

**Email:** feliciosx@gmail.com

Please include the following information whenever possible:

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Suggested mitigation (if available)

Responsible disclosure is appreciated. Reported vulnerabilities will be reviewed and addressed as appropriate.
